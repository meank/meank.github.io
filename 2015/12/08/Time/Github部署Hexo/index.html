Github部署Hexo

发表于 2015-12-08   |   分类于 闲谈

一：下载hexo

hexo需要两个依赖项：

  • Node.js
  • Git

安装完上面两个依赖项之后，运行以下命令进行安装（右键，点击git bash）：

  • $ npm install -g hexo-cli

二：部署在本地

进入本地文件夹，运行以下命令：

  • $ hexo init
  • $ cd
  • $ npm install

三：将hexo部署在github上：

首先需要注册一个github账号，生成一个类似于xxxxxx.github.io的域名（请自行百度）

打开folder/_config.yml,找到如下字段（一般在文件最后面）：

  • # Deployment
  • ## Docs: http://hexo.io/docs/deployment.html
  • deploy:

将以上内容修改为（将repository的内容修改为自己的项目https地址）：

  • # Deployment
  • ## Docs: http://hexo.io/docs/deployment.html
  • deploy:
    • type: git
    • repository: https://github.com/meank/meank.github.io.git
    • ranch: master

运行以下命令后，便可以在 http://xxx.github.io/ 查看自己的Blog

  • hexo generate
  • hexo deploy (会提示输入github的账户名和密码)
